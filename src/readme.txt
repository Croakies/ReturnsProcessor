Returns Processor Documentation

Purpose:
The Returns Processor application is a Java/SWT based application to aid in the
processing of return product and product counting in general.  The application
maintains a local database of product and UPC information that is updated
whenever the master-UPC information is updated.

Installation:
Ensure that you have a Java Runtime Environment installed on your computer.
JRE's may be downloaded from: http://www.oracle.com/technetwork/java/javase/downloads/index.html
Unpack the contents of the .zip file into a folder on your computer (the .zip is
where this readme is located at initially).
